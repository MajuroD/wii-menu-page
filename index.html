位置

馬紹爾群島位於大洋洲，在太平洋中間。

它是一個很小，但是很特別的島國。

大洋洲 = Oceania 太平洋 = Pacific Ocean 中間 = Middle 島國 = Island Country

環礁和小島

馬紹爾群島是由很多小島和環礁組成的國家。

這些島嶼讓馬紹爾群島的地理很特別。

小島 = Small Islands 環礁 = Atolls 組成 = Made Up Of 國家 = Country

馬朱羅

馬朱羅是馬紹爾群島的首都。

它是一個很長、很窄的島，所以主要只有一條路。

首都 = Capital 很長 = Very Long 很窄 = Very Narrow 一條路 = One Road

海洋生活

因為馬紹爾群島被海洋包圍，所以海洋對生活非常重要。

有時候一邊可以看到潟湖，另一邊可以看到大海。

海洋 = Ocean 潟湖 = Lagoon 大海 = Open Ocean 自然 = Nature

航海

以前的馬紹爾人很會航海。

他們會看星星、海流、風和波浪來找到方向。

航海 = Navigation 星星 = Stars 海流 = Ocean Currents 方向 = Direction

核試驗

後來，美國在馬紹爾群島做了很多核試驗。

這是馬紹爾群島歷史中很痛苦的一部分。

核試驗 = Nuclear Testing 痛苦 = Painful 土地 = Land 健康 = Health
返回歷史 下一個

Project 4.1

Project 4.1 是核試驗之後的醫學研究。

很多人覺得這件事情不公平，因為馬紹爾人沒有被清楚地告知。

醫學研究 = Medical Study 不公平 = Unfair 告知 = Informed 同意 = Consent

堅強

這段歷史很難過，但是也很重要。

它讓我更珍惜我的文化，也讓我覺得馬紹爾人很堅強。

歷史 = History 文化 = Culture 珍惜 = Value 堅強 = Strong

抓關鍵字

我聽中文的時候，不一定每一個字都聽得懂。

所以我會先抓重要的關鍵字，幫助我了解大概的意思。

今天 = Today = rainin 明天 = Tomorrow = ilju 哪裡 = Where = ia 為什麼 = Why = etke 吃飯 = Eat = ṃōñā 上課 = Class = kilaaj

問題詞

問題詞可以幫助我知道對方想問什麼。

聽到問題詞以後，我比較容易知道問題的方向。

誰 = Who = wōn 什麼 = What = ta 哪裡 = Where = ia 什麼時候 = When = nāāt 為什麼 = Why = etke 怎麼 = How = ewi

語言和溝通

我覺得語言不只是文法。

語言也是一種溝通方式，讓我和家人、文化、家鄉，還有台灣連在一起。

家人 = Family = baamle 文化 = Culture = manit 家鄉 = Hometown = aelōñ eo aō 台灣 = Taiwan = Taiwan 溝通 = Communication = kōnono

Iakwe

Iakwe 是馬紹爾語的「你好」。

它不只是打招呼，也有愛和尊重的意思。

你好 = Hello = Iakwe 愛 = Love = Iakwe 尊重 = Respect = Kautiej 文化 = Culture = Manit
